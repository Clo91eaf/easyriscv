# Easy RISC-V

https://dramforever.github.io/easyriscv

## Build

[Pandoc] is required. The build command is:

[Pandoc]: https://pandoc.org/

```
pandoc --toc --template=template.html --lua-filter=filter.lua -o index.html index.md
```
